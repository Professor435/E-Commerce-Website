# E-Commerce-Website

NEXUS is a cutting-edge, immersive e-commerce platform designed for tech enthusiasts and digital natives. Featuring a bold dark aesthetic with vibrant neon accents, the site delivers a futuristic shopping experience through smooth GSAP animations, interactive product grids, and a fully functional real-time shopping cart. The responsive design ensures seamless browsing across all devices, while micro-interactions—including a custom cursor, hover effects, and scroll-triggered reveals—create an engaging, premium feel that transforms online shopping into an exploration of tomorrow's technology.
